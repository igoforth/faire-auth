Deploy Faire Auth to Cloudflare Workers

Click the button below to deploy the Cloudflare Workers template to your Cloudflare account.

This will:

  • Clone the template into a new repo on your GitHub account
  • Provision a D1 database and KV namespace
  • Configure Workers Builds (CI/CD)
  • Deploy your auth worker

You'll need to set FAIRE_AUTH_SECRET and optionally configure GitHub/Google OAuth credentials after deployment.
